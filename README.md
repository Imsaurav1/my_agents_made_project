# Clock and Counter

A simple website that displays the current time and a counter that can be started, stopped, and paused.

## Usage

1. Open `index.html` in a web browser.
2. Click the "Start" button to start the counter.
3. Click the "Stop" button to stop the counter.
4. Click the "Pause" button to pause the counter.

## Notes

* This project uses HTML, CSS, and JavaScript.
* The clock updates every second.
* The counter increments every second when started.
* The website is responsive and accessible.
